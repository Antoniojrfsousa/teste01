# Portfólio Pessoal — AJ Feitosa

Bem-vindo ao meu portfólio! Este projeto foi desenvolvido para apresentar minha trajetória como aspirante a Desenvolvedor Full Stack, meus projetos, habilidades e formas de contato.

## 🚀 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível das páginas
- **CSS3** — Estilização responsiva e moderna com Flexbox e Grid
- **JavaScript** — Interatividade, manipulação de DOM e menu responsivo

## 📁 Estrutura do Projeto

- `index.html` — Página principal com seções de Sobre, Projetos e Contato
- `css/style.css` — Estilos personalizados com responsividade
- `img/` — Imagens utilizadas no site (logo, projetos, fundo)
- `js/app.js` — Scripts para funcionalidades como menu mobile

## 📱 Funcionalidades

- Layout responsivo para diferentes tamanhos de tela
- Menu hamburguer para navegação em dispositivos móveis
- Cards de projetos com hover animado
- Formulário de contato funcional e estilizado
